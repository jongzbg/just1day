System Architecture:

Frontend:
- Next.js (App Router)
- Component-based structure

Backend:
- NestJS (modular architecture)
- Services:
  - Auth
  - User
  - Post
  - Feed
  - Notification

Database:
- PostgreSQL

Supporting Services:
- Redis (cache + queue)
- Cloudflare R2 (image storage)

Principles:
- Separation of concerns
- Scalable design
- API-first approach
